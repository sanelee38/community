##遇到的问题：

1.无法启动项目，显示没有对应的bean。
        
        原因：发现是包名引用错误。
        
        解决方法：重新引用后成功启动项目

2.项目启动成功后，用户登录数据无法返回到h2数据库。

        原因：数据库数据项目与返回数据项目不匹配。
        
        解决：添加数据项目，解决问题。
